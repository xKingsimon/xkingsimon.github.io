Berners Lee

Vor 1990, bevor das HTTPS Protokol erfunden wurde, war kein Datenverkehr zwischen Internetnutzern verschlüsselt, Hacker oder einfach nur Interessierte konnten einfach den Datenverkehr anzapfen und mitlesen, heutzutage mit verschlüsselten Zahlungen, Nachrichten und Nutzerdaten ist ein solch unsicheres Internet unvorstellbar! 1990 schlug der britische Physiker und Informatiker Berners-Lee seinem Arbeitgeber CERN (Europäische Organisation für Kernforschung) ein Prinzip, das auf Hypertext beruht, vor. Ende 1990 und in den Folgejahren veröffentlichte er:

  • Die Seitenbeschreibungssprache HTML
  • Das Transferprotokoll HTTPS
  • Die URL
  • Den ersten Browser WorldWideWeb
  • Und die erste Website cern.ch

Durch seine Erfindung wurde ein bedeutender Meilenstein für die Entwicklung des Internets gesetzt. Durch sein HTTPS Protokol wurde nicht nur eine verschlüsselte Verbindung zum Server hergestellt, sondern auch die Echtheit dessen geprüft. Inzwischen ist die Verschlüsselung von Websiten so prominent, dass moderne Browser auf unsichere Verbindungen bei HTTP Verbindungen hinweisen.
